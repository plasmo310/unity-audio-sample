## unity-audio-sample
* UnityAudio関連の処理をまとめたサンプルプロジェクトになります。
* 基本機能からAudioMixerの活用、スペクトラムアナライザの表示まで用意しています。

### バージョン
* Unity
  * 2022.3.16f1
